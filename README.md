## 🎯 Overview

An AI-powered real-time emotion detection system that identifies human facial expressions using Deep Learning and Computer Vision. Built with TensorFlow, OpenCV, and Flask.

**👨‍💻 Created by: Shailendra Meghwal**

## ✨ Features

- 📹 **Real-time face detection & emotion recognition**
- 🖼️ **Image upload support**
- 🎭 **7+ emotion detection** (Happy, Sad, Angry, Surprise, Fear, Disgust, Neutral)
- 📊 **Detection history tracking**
- 🎨 **Modern, responsive UI**
- ⚡ **Fast prediction with CNN model**
- 🔒 **Privacy-focused (no data stored permanently)**

## 🚀 Technologies Used

### Frontend
- HTML5, CSS3, JavaScript
- Responsive Design
- WebSocket for real-time feed

### Backend
- Python 3.8+
- Flask Web Framework
- RESTful API Architecture

### AI/ML
- TensorFlow 2.13 / Keras
- OpenCV for face detection
- Convolutional Neural Networks (CNN)
- Haar Cascade Classifier
